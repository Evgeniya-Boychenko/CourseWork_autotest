**Валидные данные (используются во всех позитивных сценариях):**

**Перечень используемых инструментов с обоснованием выбора**

1. **Python** - язык, который я изучала.
2. **pytest** - фреймворк для запуска тестов.
3. **Selenium WebDriver** - фреймворк для автоматизации  веб-тестирования, который позволяет имитировать действия пользователя в браузере
4. **MySQL** - система управления БД.
5.  **Allure Report** - инструмент для генерации визуальных HTML отчетов.
6. **Docker + Docker Compose** - для развертывания изолированного экземпляра БД MySQL
7. **GitHub** - Публичная система контроля версий для хранения кода, документации и истории изменений.
